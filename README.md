# מתמטיקה ב׳ (CLB)

חומרי לימוד אינטראקטיביים לקורס **מתמטיקה ב׳** — כלים כמותיים לקבלת החלטות ניהוליות (הסתברות ותכנון ליניארי).

מרצה: ד״ר שי צבאן · אחראי אקדמי: ד״ר אסף שריג

## תוכן

| קובץ | נושא |
| --- | --- |
| [`index.html`](index.html) | פרק 1.1 — הסתברות משותפת של מאורעות תלויים (כלל הכפל) |

כל עמוד הוא קובץ HTML עצמאי אחד, כולל פונטים משובצים, דיאגרמות אינטראקטיביות (עצי הסתברות מדורגים, דיאגרמות חיתוך, מחזור התרגום) ותרגולים עם פתרונות. אין תלות בשרת או ברשת.
